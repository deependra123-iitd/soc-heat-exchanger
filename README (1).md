Self-Organised Criticality in Heat Exchangers

Why this project?

Industrial heat exchangers degrade slowly and then suddenly lose
performance.
This project shows fouling behaves like a "complex system at criticality".

1. Small events occur frequently.
2. Large failures are rare but inevitable.
3. Gaussian models underestimate risk.

Objectives:

1. Apply complexity science to heat exchanger fouling.
2. Simulate avalanche dynamics using a lattice model.
3. Analyse avalanche size distributions.
4. Study tube connectivity effects.
5. Extract engineering insights.

Simulation Overview:

  Parameter      Value
  Grid size      15 × 15
  Threshold      zc = 4
  Drive steps    5000
  Connectivity   k = 4 and k = 8
  Boundary       Open

Key Results:

1. System evolves to a "Self‑Organised Critical state".
2. Avalanche sizes follow "power‑law distribution".
3. Higher connectivity → larger cascades
4. Large fouling events are "rare but unavoidable".


Engineering Insights:

1. Heat exchangers operate near a critical state.
2. Preventive cleaning beats reactive maintenance.
3. Tube connectivity is a design risk parameter.
